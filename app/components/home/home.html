Administration Activities

Admin will create the trainers, schedule all the events and moniotor the different activities including:

  • Monitor open Alerts

    Alert may be created by trainer or by the admin to indicate on issue with specific event. The Admin will have list of open evens that requrie his attention. From specific alert the admin may open the event and take the needed action, such as find replacement or cancel the event.

  • Create and View Messages

    Message may be used to communicate with otehr trainers - e.g. in case need to cancel an activity or notify on repalcement. Administrator may see messages created for him or create message for others

  • Review Past/Future Events per Trainer

    The administrator can see summery of activities held by trainer in specific month and the detailed list of events on weekly/monthly basis.

  • Review weekly/Daily activiteis per Site

    The administrator can view all activities planned for specifc site on daily/weekly view. the administrator will be able to add/cancel/update existing events

Trainers Activities

Trainer need to sign-up with the user id provided by the admin, and from this point he may login to the system. trainers will be able to do several activities including:

  • Check Past/Future Events

    The trainer can see the activiites he particiapted on weekly/monthly basis. The trainer can see also future planned events for him.

  • Create Alert for coming event

    Alert may be created if trainer cannot attend specific event and need replacement. it will allow the admin to get teh needed information and look for replacement. Trainer may view the status of the alert till it is handled and closed by the Admin

  • Create and View Messages

    Message may be received from other users including the Admin. In addition Trainer may send message to otehr user. After reading the message the trainer may delete it.
