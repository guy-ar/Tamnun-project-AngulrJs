Administration

How It Works

Add Trainers

Admin will be creating the trainers via Trainer list. Trainer may sign up to the application and get access to options available for trainers.

Add Event

New Event may be a group class, or private lesson. Admin will have the option to add new Event, and decide how many occurrences it will include. Event will have specific date and time and dedicated trainer. The admin can open calendar to see all existing events in the same month in order to find the best suitable time slot.

Create Alert

Alert can be created by Admin or by Trainer on specific event, in order to provide notification for action, for example wen trainer may not attend the event and looking for replacement. Open alerts will be tracked both by Admin and trainer, till they are resolved

Send Message

This option will be part of next release, in order to communicate between trainers and Admin

Administration Activities

Admin will create the trainers, schedule all the events and monitor the different activities including:

  • Monitor open Alerts

    Alert may be created by trainer or by the admin to indicate on issue with specific event. The Admin will have list of open events that require his attention. From specific alert the admin may open the event and take the needed action, such as find replacement or cancel the event.

  • Create and View Messages
    Future Release
  • Review Past/Future Events per Trainer

    The administrator can see summery of activities held by trainer in specific month and the detailed list of events on weekly basis.

  • Review weekly activities per Site

    The administrator can view all activities planned for specific site on daily/weekly view. the administrator will be able to add/cancel/update existing events

Trainers Activities

Trainer need to sign-up with the user id provided by the admin, and from this point he may login to the system. trainers will be able to do several activities including:

  • Check Past/Future Events

    The trainer can see the activities he participated on weekly basis. The trainer can see also future planned events for him.

  • Create Alert for coming event

    Alert may be created if trainer cannot attend specific event and need replacement. it will allow the admin to get the needed information and look for replacement. Trainer may view the status of the alert till it is handled and closed by the Admin
